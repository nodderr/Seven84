# Seven.84 Band Website

A world-class band portfolio for **Seven.84**, a Hindustani-Bollywood-Fusion band from MIT Manipal.

## Features
- **Dynamic SPA**: Built with Vanilla JS and Vite for speed.
- **Custom Branding**: Features the signature **Samarkan** font logo.
- **Responsive Design**: Mobile-first, optimized for all screens.
- **Performance Focused**: Minimal dependencies, optimized assets.

## Tech Stack
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom Properties)
- **Deployment**: Optimized for GitHub Pages / Vercel

## Local Development
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## Deployment
This project is configured for easy deployment to **GitHub Pages**. 
Pushes to the `main` branch will automatically trigger a build and deploy via GitHub Actions.
